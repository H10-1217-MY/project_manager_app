# \# 📂 Project Manager App

# 

# \## 概要

# 

# フォルダ・ファイルを「プロジェクト単位」で管理するためのデスクトップアプリケーションです。

# ローカル環境のファイル整理を効率化し、一覧・登録・参照を一元化します。

# 

# \---

# 

# \## 🎯 主な機能

# 

# \* 📁 プロジェクト単位でフォルダ管理

# \* 📝 プロジェクト情報の登録・更新

# \* 🔍 ファイルの一覧表示・参照

# \* 💾 JSONベースの軽量データ管理

# 

# \---

# 

# \## 🧱 システム構成

# 

# ```

# project\_manager\_app/

# ├── core/     # ビジネスロジック

# ├── models/   # データ構造

# ├── ui/       # UI（画面）

# ├── utils/    # 共通処理

# ├── main.py   # エントリーポイント

# ```

# 

# \---

# 

# \## 🚀 起動方法

# 

# ```bash

# python main.py

# ```

# 

# \---

#
