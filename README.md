# \# 📂 Project Manager App

# 

# \## 🧭 概要

# 

# ローカル環境のフォルダ・ファイルを「プロジェクト単位」で管理するデスクトップアプリケーションです。

# ファイル整理を効率化し、視認性と管理性を向上させます。

# 

# 

# \## 🎯 主な機能

# 

# \* 📁 プロジェクト単位のフォルダ管理

# \* 📝 プロジェクト情報の登録・更新

# \* 🔍 ファイル一覧の表示・参照

# \* 💾 JSONベースの軽量データ管理

# 

# \---

# 

# \## 🔄 処理の流れ

# 

# 1\. プロジェクト作成

# 2\. フォルダ・ファイル登録

# 3\. 情報をJSONで管理

# 4\. UIから一覧・参照

# 

# \---

# 

# \## 🧱 ディレクトリ構成

# 

# ```

# project\_manager\_app/

# ├── core/     # ビジネスロジック

# ├── models/   # データ構造

# ├── ui/       # UI（画面）

# ├── utils/    # 共通処理

# ├── main.py   # エントリーポイント

# ```

# 

# \---

# 

# \## 🚀 起動方法

# 

# ```bash

# python main.py

# ```
